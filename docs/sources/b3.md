# Fontes B3

Dados de contratos futuros da B3 (Bolsa de Valores).

---

## Ajustes Diarios (B3)

Ajustes diarios de contratos futuros negociados na B3.

### Como usar

1. No dock agrobr, expanda a categoria **Mercado**
2. Selecione **Ajustes Diarios**
3. Preencha os parametros abaixo
4. Clique em **Buscar Dados**

### Parametros

| Parametro | Tipo | Obrigatorio | Descricao |
|-----------|------|:-----------:|-----------|
| data | Data | Sim | Data do pregao |
| contrato | Texto | Nao | Codigo do contrato (ex: BGI, SFI, SOJ) |

### Capacidades

- [x] Saida tabular
- [ ] Saida geoespacial
- [x] Filtro temporal
- [ ] Filtro por bounding box
- [ ] Join municipal

---

## Historico (B3)

Historico de precos de contratos futuros negociados na B3.

### Como usar

1. No dock agrobr, expanda a categoria **Mercado**
2. Selecione **Historico**
3. Preencha os parametros abaixo
4. Clique em **Buscar Dados**

### Parametros

| Parametro | Tipo | Obrigatorio | Descricao |
|-----------|------|:-----------:|-----------|
| contrato | Texto | Sim | Codigo do contrato (ex: BGI, SFI, SOJ) |
| inicio | Data | Sim | Data inicial do periodo |
| fim | Data | Sim | Data final do periodo |
| vencimento | Texto | Nao | Codigo de vencimento |

### Capacidades

- [x] Saida tabular
- [ ] Saida geoespacial
- [x] Filtro temporal
- [ ] Filtro por bounding box
- [ ] Join municipal

---

## Notas

- Para futuros agro, os principais contratos sao: BGI (Boi Gordo), SFI (Cafe), SOJ (Soja), CCM (Milho).
