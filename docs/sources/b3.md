# Fontes B3

Dados de contratos futuros da B3 (Bolsa de Valores).

---

## Ajustes Diários (B3)

Ajustes diários de contratos futuros negociados na B3.

### Como usar

1. No dock agrobr, expanda a categoria **Mercado**
2. Selecione **Ajustes Diários**
3. Preencha os parâmetros abaixo
4. Clique em **Buscar Dados**

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| data | Data | Sim | Data do pregão |
| contrato | Texto | Não | Código do contrato (ex: BGI, SFI, SOJ) |

### Capacidades

- [x] Saída tabular
- [ ] Saída geoespacial
- [x] Filtro temporal
- [ ] Filtro por bounding box
- [ ] Join municipal

---

## Histórico (B3)

Histórico de preços de contratos futuros negociados na B3.

### Como usar

1. No dock agrobr, expanda a categoria **Mercado**
2. Selecione **Histórico**
3. Preencha os parâmetros abaixo
4. Clique em **Buscar Dados**

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| contrato | Texto | Sim | Código do contrato (ex: BGI, SFI, SOJ) |
| inicio | Data | Sim | Data inicial do período |
| fim | Data | Sim | Data final do período |
| vencimento | Texto | Não | Código de vencimento |

### Capacidades

- [x] Saída tabular
- [ ] Saída geoespacial
- [x] Filtro temporal
- [ ] Filtro por bounding box
- [ ] Join municipal

## Notas

- Para futuros agro, os principais contratos são: BGI (Boi Gordo), SFI (Café), SOJ (Soja), CCM (Milho).
