# Terras Indigenas (FUNAI)

Terras indigenas demarcadas no Brasil.

## Como usar

1. No dock agrobr, expanda a categoria **Fundiario**
2. Selecione **Terras Indigenas**
3. Preencha os parametros abaixo
4. Clique em **Buscar Dados**

## Parametros

| Parametro | Tipo | Obrigatorio | Descricao |
|-----------|------|:-----------:|-----------|
| uf | UF | Nao | Filtrar por estado |
| fase | Texto | Nao | Fase da demarcacao |
| bbox | Bounding Box | Nao | Filtrar por area geografica |

## Capacidades

- [x] Saida geoespacial — marque "Saida geoespacial" para obter geometrias
- [ ] Filtro temporal
- [x] Filtro por bounding box
- [ ] Join municipal

## Notas

- Inclui todas as fases de demarcacao (em estudo, delimitada, declarada, homologada, regularizada).
